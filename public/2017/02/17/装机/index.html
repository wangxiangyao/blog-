装机

发表于 2017-02-17 | 分类于 生活

终于有了台属于自己的电脑，i5-7500 + 1060，很开心，哈哈。而作为专业的前端人员，要开始工作，还需要很多准备。
以此记录这次装机过程，留作备份。

一些基本的就不说了，从前端说起。

必备（不分先后顺序）

  • chrome
    直接网上搜，安装之后，顺便去谷歌商店下postman。O(∩_∩)O
  • 编辑器（开发环境）
    sublime3 不解释，可以直接去官网下载。
    贴个链接——sublime3资料-插件，安装，许可证等

必备暂时就这俩吧，还没想起来，想起来再补充。

常用

  • 词典
    目前用的有道词典
  • 笔记软件
    印象笔记(笔记软件) + 马克飞象（配套的markdown编辑器）印象笔记可以免费试用，只是权限少，马克飞象有试用期，过了就得掏钱。（好吧，我都掏钱了。）
  • 记事本
    NotePad++
  • 蓝灯
    翻墙利器，浏览墙外的官网，下载个包，能用到，当然，不止如此哦。
    贴个链接——蓝灯的github
  • FSCapture
    屏幕操作软件，截屏，录屏，等等
  • abodeReader
    pdf阅读
  • Everything
    本地搜索工具
  • flux
    护眼的小软件，刚开始用着不习惯，后来感觉，恩，还不错呢
  • picpick
    调色板，，，好吧，其实我做常用的，只是用它取色而已。
  • XMind
    用来整理思路很有用，听说最近在招聘？(⊙o⊙)

好了，常用的小软件，大概就这些，下面就是正题了！

开发相关

其实，现在仔细想想，对于前端来说，工具，说白了就是各种包

  • npm（node）
    node自带npm很方便，上Node官网，下一个Node,然后，安装就行了，目前我用的6.9.5稳定版本。
    当然，还有nvm，然而，配置有点麻烦（其实是我忘了怎么配了o(╯□╰)o），所以这台电脑，只安装了一个版本的node，如果需要别的版本，直接再安一个就好。（nvm再找找资料，应该有的，忘了放在哪了。）
  • git
    直接官网就行了
    贴个链接git 官网
    哪个系统就下那种git包就好。
    至于教程，我是看的廖雪峰的git学的
    • github
      这就不得不提github了，由于是新电脑，还得配秘钥，教程上都有。
      把github上的东西down下来（其实就一个博客页）

其实，有了npm之后，各种工具包就能从他上边下了，babel，webpack、gulp等等，当然，还有各种的开发包，总之有了npm，差不多啥都有了。

其他

一些我正在试着用，感觉不错的工具，有web的，有客户端的。
一些不知道咋分类的工具

  • 简书
    markdown编辑器，有朋友找不多怎么调markdown，注册个账户，然后进设置（点自己头像，反正就那一块），里边有个选项，富文本 / markdown
    ，选markdown。
    因为要在博客上写文章，又用的微软系统，没办法，用简书的编辑器吧，做的很不错呢。
    当然，简书还是一个很好的网站，上边很多文章都是有水平的。
  • fiddler
    监视http包的，很好用，但不常用
  • koala
    其实不太喜欢用Less/sass感觉没什么卵用，再加上现在css3也有变量了，更不用less/sass了。
  • mysql
    • navicat
  • nginx
    上边这俩，都不太会，但是要学，目前掌握一些皮毛，不打紧，毕竟精力有限，先学好前端，再说这些。
  • postman
    模拟http请求的，这个需要翻墙，到谷歌商店下载
  • 印象笔记·剪藏
    同样，翻墙，到谷歌商店

总结

其实，如果只是开发，4个足够了：
浏览器 + 编辑器 + npm + node
