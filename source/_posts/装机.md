## 开发相关

其实，现在仔细想想，对于前端来说，工具，说白了就是各种包
- npm（node）
node自带npm很方便，上Node官网，下一个Node,然后，安装就行了，目前我用的6.9.5稳定版本。
当然，还有nvm，然而，配置有点麻烦（其实是我忘了怎么配了o(╯□╰)o），所以这台电脑，只安装了一个版本的node，如果需要别的版本，直接再安一个就好。（nvm再找找资料，应该有的，忘了放在哪了。）
- git
直接官网就行了
贴个链接[git 官网](https://git-scm.com/)
哪个系统就下那种git包就好。
至于教程，我是看的[廖雪峰的git](http://www.liaoxuefeng.com/wiki/0013739516305929606dd18361248578c67b8067c8c017b000/)学的
  - github
这就不得不提github了，由于是新电脑，还得配秘钥，教程上都有。
把github上的东西down下来（其实就一个博客页）

其实，有了npm之后，各种工具包就能从他上边下了，babel，webpack、gulp等等，当然，还有各种的开发包，总之有了npm，差不多啥都有了。

## 其他
一些我正在试着用，感觉不错的工具，有web的，有客户端的。
一些不知道咋分类的工具

- 简书
markdown编辑器，有朋友找不多怎么调markdown，注册个账户，然后进设置（点自己头像，反正就那一块），里边有个选项，富文本 / markdown
，选markdown。
因为要在博客上写文章，又用的微软系统，没办法，用简书的编辑器吧，做的很不错呢。
当然，简书还是一个很好的网站，上边很多文章都是有水平的。
- fiddler
监视http包的，很好用，但不常用
- koala
其实不太喜欢用Less/sass感觉没什么卵用，再加上现在css3也有变量了，更不用less/sass了。
- mysql
  - navicat
- nginx
上边这俩，都不太会，但是要学，目前掌握一些皮毛，不打紧，毕竟精力有限，先学好前端，再说这些。
- postman

- 印象笔记·剪藏
