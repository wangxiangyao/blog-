## 回顾

最近几个月在学习什么呢？

- html + css + javascript
	- bootstrap + jQ + animate.js + WOW.js 等等，用过的框架大概就这几个
	- html + css 学习了html4.01、html 5的语法，一些很有趣的新特性，确实有很大学问，比如，原生css变量，语义化标签，css动画(牵扯到渲染引擎，有事件写)、多媒体...等等。

		> 以上知识来自——《javascript高级程序设计》，还有老师的帮助。多谢！
